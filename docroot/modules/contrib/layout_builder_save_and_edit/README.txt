INTRODUCTION
------------

This Module adds 'Save and Edit Layout' button in layout builder form page which
will save site builders/content editors time by redirecting it to same page in
which they can see there changes and can add extras features for there layout.


REQUIREMENTS
------------

This module requires Core Layout Builder.


INSTALLATION
------------

Install the Module as you would normally install a contributed Drupal
module. Visit https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
--------------

This module does not have any configuration.
